# 政府材料价格看板

政务数据材料价格查询与可视化平台，基于 Elasticsearch 数据源，支持多维度筛选、价格趋势分析、涨跌幅监控。

## 技术栈

**后端**：Python + FastAPI + Elasticsearch  
**前端**：Vue 3 + Vite + ECharts  
**数据源**：Elasticsearch（政务价格数据库）

## 快速启动

```bash
# 克隆后一键启动（自动安装依赖）
./start.sh
```

启动后访问：
- 前台：http://localhost:5300
- API：http://localhost:5200
- API 文档：http://localhost:5200/docs

## 功能特性

### 筛选查询
- **关键词搜索**：支持产品名称模糊匹配
- **省份筛选**：支持全国各省份下拉选择
- **高级筛选**：城市、区县、价格区间精细筛选
- **最近搜索**：自动记录并快速回溯搜索历史

### 数据展示
- **表格视图**：产品名称 / 规格·城市 / 价格 / 含税价 / 单位 / 日期
- **图表视图**：价格趋势折线图 + 涨幅/跌幅排行榜
- **行情风格**：主价格放大显示 + 含税价灰色次要展示 + 涨跌箭头百分比
- **视觉层级**：主信息深色大字号 / 次信息灰色 / 单位标签化 / 数值居右

### 数据指标
- **涨幅榜 / 跌幅榜**：基于近两期均价变化率排名
- **月度趋势**：柱状图（数据条数）+ 折线图（月均价）
- **品类统计**：各品类文档数量分布

## API 接口

| 接口 | 说明 |
|------|------|
| `GET /api/search` | 搜索查询，支持关键词/省份/城市/区县/价格范围分页 |
| `GET /api/stats/overview` | 全局概览：数据总量、省份/城市/品类/单位分布 |
| `GET /api/stats/top-products` | 品类统计排行 |
| `GET /api/filter-options` | 筛选器下拉选项：省份、城市、区县、单位 |
| `GET /api/stats/price-trend` | 价格趋势，按月聚合 |
| `GET /api/stats/price-change` | 涨跌幅统计，近两月均价对比 |

## 数据说明

- **ES 索引**：`gov_price_index`
- **数据截止**：支持历史数据查询（部分省份数据截至 2026-02）
- **价格字段**：`price`（原价）、`tax_price`（含税价）、`avg_price`（品类均价）
- **时间范围**：价格趋势默认展示近 12 个月数据
