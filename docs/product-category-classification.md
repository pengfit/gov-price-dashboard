## 分类算法说明

```
规则优先级：按关键词长度降序匹配（优先匹配更长词）
分类逻辑：
  1. 提取产品名称 breed
  2. 与23个一级分类关键词列表逐个匹配
  3. 命中即归类，未命中归为"其他"
  4. 可按二级分类（如：钢管 → 无缝钢管/焊接钢管）进一步细分
```
