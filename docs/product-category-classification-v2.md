## 分类算法

```python
CLASSIFY_RULES = [
    # (关键词列表, 分类名)
    (['钢管','焊接钢管','镀锌管',...], '管材管件'),
    (['电缆','电力电缆','控制电缆',...], '电气材料'),
    # ... 共32条一级规则
]

def classify(name: str) -> str:
    name_lower = name.lower()
    for keywords, category in CLASSIFY_RULES:
        for kw in keywords:
            if kw.lower() in name_lower:
                return category
    return '其他'
```

### 匹配优先级
1. **关键词精确匹配优先**（如 `PP-R` 优先于 `PPR`）
2. **长词优先于短词**（如 `热轧带肋钢筋` 优先于 `钢筋`）
3. **未匹配产品归入「其他」**

### 数据覆盖率说明
- Top2000 类别覆盖 **76.4%** 文档
- 另有 **23.6%**（约43万条）落在 Top2000 以外的极长尾产品
- 长尾产品多为特殊规格、地方项目产品，每种1-5条，难以结构化归类
